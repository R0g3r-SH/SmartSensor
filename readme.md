**Conections:**


**SD module**
-SD  (GND) -- Ground  
-SD  (3.3v) --3v  
-SD  (MISO) --D6  
-SD (MOSI) -- D7  
-SD  (SCK/clock) -- D5  
-SD  (CS) -- D8  

**DHT**
pin_1 -> GND  
pin_2 DATA -> RX  
pin_3 -> 3.3v  

**OLED**
pin_1 -> GND  
pin_2 -> 3.3v  
pin_3 SCK -> D1  
pin_3 SDA -> D2  

**CLOCK** 
pin_1 -> GND  
pin_2 -> 3.3v  
pin_3 SCK -> D1  
pin_3 SDA -> D2  

**BTN1 PULL UP**  
pin -> D3  

**BTN2 PULL UP**  
pin -> D4
